Dev Rules
✕Never use browser btoa() for GitHub pushes — Python from Terminal only
✕Never replace renderLibrary() — sacred localStorage version
✕Never put onclick with quotes inside JS template literal HTML strings — use data attributes
✓Cloud Library is a separate overlay — loadCloudLibrary() only, never mixed with renderLibrary
✓addToLib auto-uploads to Supabase via Railway proxy /publish
✓Proxy fetches BFL signed URLs server-side — never send base64 from browser
✓Use mirage-pwa.vercel.app not mirageicons.com for fresh deploys
✓personas.json in repo root — 100 icons for Persona Builder and Studio
